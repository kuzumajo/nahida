# 语法

跳转指令

<!-- prettier-ignore -->
```markdown
<!-- 进入下一章节（直接结束当前章节） -->
[next](./chapter-2.md)
[next](#end)

<!-- 引入目标内容（仅跳转内容，结束了就回来） -->
[jump](./choice-2.md)
```

控制台指令

<!-- prettier-ignore -->
```markdown
<!-- 显示大字 -->
![c](#title "海内存知己，天涯若比邻")

<!-- 隐藏控制台 -->
![c](#hide)

<!-- 显示控制台 -->
![c](#show)

<!-- 等待若干秒（毫秒） -->
![c](#wait "2000")
```
